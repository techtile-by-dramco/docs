## Purpose

Single page to track daily operational health and where to go next for action.

## Daily Status Areas

- Access status:
  - VPN availability and remote reachability.
- Network and timing status:
  - key infrastructure host reachability and sync readiness.
- Tile fleet status:
  - online/offline tiles, maintenance targets, and power-control actions.
- Experiment readiness:
  - pre-run checks and subsystem availability.
